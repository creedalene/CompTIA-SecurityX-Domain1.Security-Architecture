### Overview
These are my personal study notes and reference materials for **CompTIA SecurityX (CAS-005) Domain 1.0: Governance, Risk, and Compliance**. I focus heavily on practical application, especially in defense and enterprise environments.

### Repository Purpose
I built this to help senior cybersecurity professionals and defense contractors prepare for the CAS-005 exam while also serving as a living reference for real-world security architecture work. The material emphasizes how governance, risk, and compliance decisions actually get made at the architect and assessor level.

### Domain 1.0 Coverage
This domain makes up approximately **25%** of the SecurityX exam and focuses on translating organizational security requirements into effective governance structures, risk management practices, and compliance strategies.

### Key Topics
- Security program documentation and the governance hierarchy (Policy → Standard → Procedure → Guideline)
- Governance pyramids, RACI matrices, and program management
- DoD cybersecurity frameworks (RMF, STIGs, CUI, etc.)
- Zero Trust Architecture (NIST SP 800-207 + DoD ZT RA)
- Change & Configuration Management
- GRC tools and data governance across environments
- Risk management, threat modeling, and emerging AI security challenges

### DoD / Defense Focus
I place strong emphasis on:
- Risk Management Framework (RMF) per DoDI 8510.01
- DISA STIGs and IAVM processes
- NIST SP 800-53 / 800-171 implementation
- CUI protection requirements (DoDI 5200.48)
- eMASS workflows and continuous monitoring
